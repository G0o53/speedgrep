# speedgrep

A no‑nonsense, literal‑string grep clone built in Rust.

- 🔢 Shows line numbers
- 📦 Tiny binary

## 🚀 We Beat Ripgrep

On a 1,000,000‑line test file:

- **⚡ speedgrep:** 1.460 seconds  
- **🦀 ripgrep:** 8.618 seconds

#### Note: Speedgrep is a speed demon, while ripgrep is a regex engine.

Minimalism wins.
